A modern React-based AI News application that fetches real-time news and uses Hugging Face models to generate simplified, reader-friendly summaries.

## Why JavaScript (JSX) Only?

This project intentionally uses **JavaScript instead of TypeScript** to:

- Improve readability for recruiters and beginners
- Reduce configuration complexity
- Avoid unnecessary type overhead for a frontend-focused project
- Ensure faster onboarding and easier maintenance

TypeScript files generated by AI tools were fully converted to JavaScript to maintain a **clean, consistent codebase**.

## Features
- Real-time news fetching using News API
- AI-powered news summarization using Hugging Face
- Clean UI with reusable React components
- Environment-based API key management
- Fully responsive design

## Tech Stack
- React (Vite)
- JavaScript (ES6+)
- Hugging Face Inference API
- News API
